📖 While Loop ဆိုတာ ဘာလဲ?

While loop ဆိုတာ condition true ဖြစ်သရွေ့ code block ကို ထပ်ခါထပ်ခါ execute လုပ်တဲ့ loop တစ်မျိုးပါ။ For loop လို iterations အရေအတွက် သိစရာ မလိုဘူး - condition false ဖြစ်တဲ့အထိ run နေမယ်။

နေ့စဉ်ဘဝမှာ ဥပမာများ:

  • မိုးရွာနေသရွေ့ အိမ်မှာ နေပါ (condition: မိုးရွာနေတာ)
  • လမ်း ရှင်းနေသရွေ့ မောင်းပါ (condition: လမ်းရှင်းတာ)
  • Password မှားနေသရွေ့ ထပ်မေးပါ (condition: password မှန်/မမှန်)
  • Balance မလုံလောက်သရွေ့ ငွေထုတ်လို့ မရဘူး (condition: balance check)

အဲ့ဒီလို scenarios တွေမှာ while loop က perfect!

For vs While - အဓိက ခြားနားချက်

Aspect For Loop While Loop
အသုံးပြုရန် Iterations သိချင်တဲ့အခါ Condition အပေါ် မူတည်တဲ့အခါ
Structure for (init; condition; increment) while (condition)

🔹 Basic While Loop Syntax

While loop ရဲ့ basic structure:

Java 
while (condition) {
    // condition က true ဖြစ်သရွေ့ run မယ်
    statements;
    // condition ကို update လုပ်ဖို့ မမေ့ပါနဲ့!
}

အရေးကြီးတဲ့ အချက်များ:

  • Condition check: Loop body run မီ condition ကို စစ်တယ် (pre-check)
  • Update variable: Loop body ထဲမှာ condition ကို update လုပ်ရမယ် (infinite loop ဖြစ်မှာ မဟုတ်ရင်)
  • Zero iterations: Initial condition က false ဆိုရင် တစ်ကြိမ်မှ မ run ဘူး
  • Boolean condition: Condition က boolean expression ဖြစ်ရမယ် (true/false)

Flow Chart

1. Check condition → true?
   ├─ Yes: Execute loop body
   │         ↓
   │   Update variables
   │         ↓
   │   Go back to step 1
   │
   └─ No: Exit loop, continue program

🔹 Simple Example: Count to 10

While loop နဲ့ 1 ကနေ 10 အထိ ရေတွက်ခြင်း:

Java 
public class WhileCount {
    public static void main(String[] args) {
        int i = 1;  // Initialize
        
        while (i <= 10) {  // Condition
            System.out.println(i);
            i++;  // Update (အရေးကြီး!)
        }
        
        System.out.println("Done!");
    }
}

🔹 Input Validation - While Loop ရဲ့ အင်အား!

User က valid input မထည့်သမျှ ထပ်မေးဖို့ while loop က perfect:

Example 1: Password Validation

Java 
import java.util.Scanner;

public class PasswordCheck {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String correctPassword = "java123";
        String userInput = "";
        int attempts = 0;
        int maxAttempts = 3;
        
        System.out.println("=== Password Login System ===");
        
        while (!userInput.equals(correctPassword) && attempts < maxAttempts) {
            System.out.print("Enter password: ");
            userInput = scanner.nextLine();
            attempts++;
            
            if (!userInput.equals(correctPassword)) {
                int remaining = maxAttempts - attempts;
                if (remaining > 0) {
                    System.out.println("❌ Wrong password! " + remaining + " attempts left.");
                } else {
                    System.out.println("❌ Account locked! Too many failed attempts.");
                }
            }
        }
        
        if (userInput.equals(correctPassword)) {
            System.out.println("✓ Login successful! Welcome!");
        }
        
        scanner.close();
    }
}

🔹 Do-While Loop

Do-while loop က while loop နဲ့ ဆင်တယ်၊ ဒါပေမယ့် condition ကို အဆုံးမှာ စစ်တယ်။ ဆိုလိုတာက အနည်းဆုံး တစ်ကြိမ် run မယ်!

Java Syntax 
do {
    // Code ကို အရင် run တယ်
    statements;
} while (condition);  // နောက်မှ condition စစ်တယ်

While vs Do-While:

  • While: Condition အရင်စစ် → true ဖြစ်မှ run (0 or more times)
  • Do-While: အရင် run → နောက် condition စစ် (1 or more times)

Example: Menu (Must Show At Least Once)

Java 
import java.util.Scanner;

public class DoWhileMenu {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String response;
        
        do {
            System.out.println("\n=== Welcome to Our Service ===");
            System.out.println("We're here to help you!");
            System.out.println("Have a great day!");
            
            System.out.print("\nDo you want to see this again? (yes/no): ");
            response = scanner.nextLine().toLowerCase();
            
        } while (response.equals("yes"));
        
        System.out.println("Goodbye!");
        scanner.close();
    }
}

Comparison Example

Java 
// While loop
int x = 10;
while (x < 5) {
    System.out.println("This never prints!");  // ဘာမှ မထွက်ဘူး
}

// Do-While loop
int y = 10;
do {
    System.out.println("This prints once!");  // တစ်ကြိမ် ထွက်တယ်
} while (y < 5);

🔹 Infinite Loops - အန္တရာယ်!

Infinite loop ဆိုတာ condition က ဘယ်တော့မှ false မဖြစ်လို့ loop က အဆုံးမရှိ run နေတဲ့ အခြေအနေပါ:

❌ Common Infinite Loop Mistakes

WRONG! 
// Mistake 1: Forgot to update
int i = 1;
while (i <= 10) {
    System.out.println(i);
    // i++ ထည့်ဖို့ မေ့သွားတယ်!
}

// Mistake 2: Wrong update direction
int j = 1;
while (j <= 10) {
    System.out.println(j);
    j--;  // Decreasing instead of increasing!
}

// Mistake 3: Always true condition
while (true) {
    System.out.println("Forever!");
    // No break statement!
}

// Mistake 4: Wrong comparison
int count = 0;
while (count != 10) {
    System.out.println(count);
    count += 2;  // 0,2,4,6,8,10,12... ဘယ်တော့မှ 10 မတူဘူး!
}

✅ How to Avoid Infinite Loops

  • Always update: Loop variable ကို loop body ထဲမှာ update လုပ်ပါ
  • Check direction: Increment/decrement က condition နဲ့ ညီရမယ်
  • Use break: Intentional infinite loop မှာ break condition ထည့်ပါ
  • Test carefully: Small values နဲ့ test လုပ်ပါ (100 အစား 5)
  • Add counter: Safety counter ထည့်ပြီး maximum iterations limit လုပ်ပါ

Intentional Infinite Loop (With Break)

Java - Correct Usage 
import java.util.Scanner;

public class ServerSimulation {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // Server က ပိတ်ဖို့ command မရသမျှ run နေမယ်
        while (true) {
            System.out.print("Server> ");
            String command = scanner.nextLine();
            
            if (command.equals("exit")) {
                System.out.println("Shutting down server...");
                break;  // Exit the infinite loop
            }
            
            System.out.println("Processing: " + command);
        }
        
        scanner.close();
    }
}

🔹 Loop Control: Break & Continue

Break Statement

Loop ကနေ ချက်ချင်း ထွက်ဖို့:

Java 
// Example: Find first even number
int[] numbers = {1, 3, 5, 7, 8, 9, 11};
int i = 0;

while (i < numbers.length) {
    if (numbers[i] % 2 == 0) {
        System.out.println("First even number: " + numbers[i]);
        break;  // Found it, exit loop
    }
    i++;
}

// Output: First even number: 8

Continue Statement

Current iteration ကို skip လုပ်ပြီး next iteration သွားဖို့:

Java 
// Example: Print only odd numbers
int i = 1;

while (i <= 10) {
    if (i % 2 == 0) {
        i++;
        continue;  // Skip even numbers
    }
    
    System.out.println(i);  // Only odd numbers print
    i++;
}

// Output: 1, 3, 5, 7, 9

Break vs Continue:

  • break: Loop ကနေ လုံးဝထွက်သွားတယ်
  • continue: Current iteration ကို skip လုပ်ပြီး next iteration ကို ဆက်လုပ်တယ်

💻 Complete Example: Number Guessing Game

While loop, break, continue အားလုံး ပေါင်းစပ်ထားတဲ့ complete game:

Java 
import java.util.Scanner;
import java.util.Random;

public class NumberGuessingGame {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        Random random = new Random();
        
        System.out.println("=== Number Guessing Game ===");
        System.out.println("I'm thinking of a number between 1 and 100.");
        
        int targetNumber = random.nextInt(100) + 1;
        int attempts = 0;
        int maxAttempts = 7;
        boolean hasWon = false;
        
        while (attempts < maxAttempts) {
            int remaining = maxAttempts - attempts;
            System.out.println("\nAttempts remaining: " + remaining);
            System.out.print("Enter your guess: ");
            
            // Input validation
            if (!scanner.hasNextInt()) {
                System.out.println("❌ Invalid input! Please enter a number.");
                scanner.next();  // Clear bad input
                continue;  // Skip this iteration
            }
            
            int guess = scanner.nextInt();
            attempts++;
            
            // Check if out of range
            if (guess < 1 || guess > 100) {
                System.out.println("❌ Please guess between 1 and 100!");
                attempts--;  // Don't count this attempt
                continue;
            }
            
            // Check the guess
            if (guess == targetNumber) {
                hasWon = true;
                System.out.println("\n🎉 Congratulations! You won!");
                System.out.println("The number was: " + targetNumber);
                System.out.println("You guessed it in " + attempts + " attempts!");
                break;  // Exit the loop
                
            } else if (guess < targetNumber) {
                int difference = targetNumber - guess;
                
                if (difference <= 5) {
                    System.out.println("🔥 Very close! Too low!");
                } else if (difference <= 15) {
                    System.out.println("📈 Too low!");
                } else {
                    System.out.println("⬆️ Way too low!");
                }
                
            } else {
                int difference = guess - targetNumber;
                
                if (difference <= 5) {
                    System.out.println("🔥 Very close! Too high!");
                } else if (difference <= 15) {
                    System.out.println("📉 Too high!");
                } else {
                    System.out.println("⬇️ Way too high!");
                }
            }
        }
        
        if (!hasWon) {
            System.out.println("\n😢 Game Over! You ran out of attempts.");
            System.out.println("The number was: " + targetNumber);
        }
        
        System.out.println("\nThanks for playing!");
        scanner.close();
    }
}

Java 
import java.util.Scanner;

public class ATMSystem {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        // Account data
        String accountName = "Aung Aung";
        int pin = 1234;
        double balance = 50000;
        boolean isAuthenticated = false;
        int loginAttempts = 0;
        int maxLoginAttempts = 3;
        
        System.out.println("=== Welcome to A Coder Bank ATM ===");
        
        // PIN verification
        while (!isAuthenticated && loginAttempts < maxLoginAttempts) {
            System.out.print("Enter your PIN: ");
            int userPin = scanner.nextInt();
            loginAttempts++;
            
            if (userPin == pin) {
                isAuthenticated = true;
                System.out.println("✓ Login successful!");
            } else {
                int remaining = maxLoginAttempts - loginAttempts;
                if (remaining > 0) {
                    System.out.println("❌ Incorrect PIN! " + remaining + " attempts left.");
                } else {
                    System.out.println("❌ Card blocked! Too many failed attempts.");
                }
            }
        }
        
        // Main menu (if authenticated)
        if (isAuthenticated) {
            int choice = 0;
            
            while (choice != 4) {
                System.out.println("\n=== Main Menu ===");
                System.out.println("Account: " + accountName);
                System.out.println("1. Check Balance");
                System.out.println("2. Deposit");
                System.out.println("3. Withdraw");
                System.out.println("4. Exit");
                System.out.print("Enter choice (1-4): ");
                
                choice = scanner.nextInt();
                
                switch (choice) {
                    case 1:
                        // Check balance
                        System.out.println("\n=== Balance Inquiry ===");
                        System.out.println("Current Balance: " + balance + " Kyats");
                        break;
                        
                    case 2:
                        // Deposit
                        System.out.println("\n=== Deposit ===");
                        System.out.print("Enter amount to deposit: ");
                        double depositAmount = scanner.nextDouble();
                        
                        if (depositAmount > 0 && depositAmount <= 1000000) {
                            balance += depositAmount;
                            System.out.println("✓ Deposit successful!");
                            System.out.println("Deposited: " + depositAmount + " Kyats");
                            System.out.println("New Balance: " + balance + " Kyats");
                        } else {
                            System.out.println("❌ Invalid amount! (Max: 1,000,000 Kyats)");
                        }
                        break;
                        
                    case 3:
                        // Withdraw
                        System.out.println("\n=== Withdrawal ===");
                        System.out.print("Enter amount to withdraw: ");
                        double withdrawAmount = scanner.nextDouble();
                        
                        if (withdrawAmount <= 0) {
                            System.out.println("❌ Invalid amount!");
                        } else if (withdrawAmount > balance) {
                            System.out.println("❌ Insufficient balance!");
                            System.out.println("Available: " + balance + " Kyats");
                        } else if (withdrawAmount > 500000) {
                            System.out.println("❌ Daily withdrawal limit: 500,000 Kyats");
                        } else {
                            balance -= withdrawAmount;
                            System.out.println("✓ Withdrawal successful!");
                            System.out.println("Withdrawn: " + withdrawAmount + " Kyats");
                            System.out.println("Remaining Balance: " + balance + " Kyats");
                            
                            // Dispense money simulation
                            System.out.println("\nDispensing money...");
                            int[] notes = {50000, 10000, 5000, 1000};
                            double remaining = withdrawAmount;
                            
                            for (int note : notes) {
                                if (remaining >= note) {
                                    int count = (int)(remaining / note);
                                    System.out.println(note + " Kyats × " + count);
                                    remaining = remaining % note;
                                }
                            }
                        }
                        break;
                        
                    case 4:
                        System.out.println("\n=== Thank You ===");
                        System.out.println("Please take your card.");
                        System.out.println("Have a nice day!");
                        break;
                        
                    default:
                        System.out.println("❌ Invalid choice! Please select 1-4.");
                }
            }
        }
        
        scanner.close();
    }
}

🔹 Common While Loop Patterns

Pattern 1: Sentinel-Controlled Loop

Special value (sentinel) ရောက်တဲ့အထိ run:

Java 
Scanner scanner = new Scanner(System.in);
int num;

System.out.println("Enter numbers (0 to stop):");

while ((num = scanner.nextInt()) != 0) {
    System.out.println("You entered: " + num);
}

System.out.println("Done!");

Pattern 2: Flag-Controlled Loop

Boolean flag သုံးပြီး control:

Java 
boolean keepRunning = true;

while (keepRunning) {
    // Do something
    
    if (someCondition) {
        keepRunning = false;  // Stop the loop
    }
}

Pattern 3: Counter-Controlled (Known Max)

Maximum iterations ရှိတဲ့ validation:

Java 
int attempts = 0;
int maxAttempts = 5;
boolean success = false;

while (!success && attempts < maxAttempts) {
    // Try something
    attempts++;
    
    if (successfulOperation) {
        success = true;
    }
}

🎯 အဓိက မှတ်သားရမည့် အချက်များ

  • While loop: Condition true ဖြစ်သရွေ့ run တယ် (0 or more times)
  • Do-while loop: အရင် run, နောက် condition စစ် (1 or more times)
  • Use while when: Iterations မသိ, condition-based, user input validation
  • Use for when: Iterations သိ, counting, arrays
  • Always update: Loop variable ကို update လုပ်ဖို့ မမေ့ပါနဲ့ (infinite loop ဖြစ်မယ်)
  • Break: Loop ကနေ လုံးဝထွက်ဖို့
  • Continue: Current iteration skip လုပ်ဖို့
  • Input validation: While loop က perfect (valid input မရသမျှ ထပ်မေး)
  • Menu systems: Do-while သုံးရင် menu ကို အနည်းဆုံး တစ်ကြိမ် ပြတယ်
  • Infinite loops: Testing နဲ့ safety counters သုံးပြီး ရှောင်ပါ
  • Boolean flags: Complex conditions အတွက် boolean variables သုံးပါ

✍️ Practice Exercise

Exercise: Factorial Calculator

While loop သုံးပြီး factorial တွက်တဲ့ program ရေးပါ:

  • User ကနေ positive number တစ်ခု input ယူပါ
  • Factorial တွက်ပါ (5! = 5 × 4 × 3 × 2 × 1 = 120)
  • Input validation လုပ်ပါ (negative numbers reject)
  • User က "exit" လို့ ရိုက်မှ ရပ်ပါ

Example:

Enter a number (or 'exit' to quit): 5
5! = 120

Enter a number (or 'exit' to quit): 6
6! = 720

Enter a number (or 'exit' to quit): exit
Goodbye!
💡 Solution ကြည့်ရန် (Try လုပ်ပြီးမှ ကြည့်ပါ)
Java 
import java.util.Scanner;

public class FactorialCalculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String input = "";
        
        System.out.println("=== Factorial Calculator ===");
        
        while (!input.equals("exit")) {
            System.out.print("Enter a number (or 'exit' to quit): ");
            input = scanner.nextLine();
            
            if (input.equals("exit")) {
                System.out.println("Goodbye!");
                break;
            }
            
            try {
                int n = Integer.parseInt(input);
                
                if (n < 0) {
                    System.out.println("Error: Please enter a positive number!");
                    continue;
                }
                
                long factorial = 1;
                int i = 1;
                
                while (i <= n) {
                    factorial *= i;
                    i++;
                }
                
                System.out.println(n + "! = " + factorial);
                
            } catch (NumberFormatException e) {
                System.out.println("Error: Invalid input!");
            }
            
            System.out.println();
        }
        
        scanner.close();
    }
}
